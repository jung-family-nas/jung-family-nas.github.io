개인폴더

개인폴더는 Nextcloud 프로그램을 통해서 관리됩니다.

다양한 프로토콜을 지원합니다.

계정 정보

  • USERNAME은 영문 이름만 사용합니다.
  • PASSWORD는 qkRnjwntpdy입니다. (바꿔주세요)

사용법 1. 웹 인터페이스

jung-family-nas.uk:9001

첫번째 사용 방법입니다. Web UI를 통해서 접속합니다.

사용법 2. WebDAV

https protocol을 사용해 윈도우/맥 등의 탐색기에서 직접 접근이 가능합니다.

2.1. Windows - WebDAV

  1. 파일 탐색기 열기
  2. “이 PC”를 마우스 오른쪽 클릭
  3. “네트워크 위치 추가” 선택
  4. “다음”을 계속 클릭하여 인터넷 주소를 묻는 화면까지 이동
  5. https://jung-family-nas.uk:9001/remote.php/dav/files/USERNAME 입력
  6. “다음” 클릭
  7. Nextcloud 사용자 이름과 암호 입력

2.1. macOS - WebDAV

  1. 파인더 열기
  2. 메뉴 바에서 “이동” 선택
  3. “서버에 연결” 선택
  4. https://jung-family-nas.uk:9001/remote.php/dav/files/USERNAME 입력
  5. “연결” 클릭
  6. Nextcloud 사용자 이름과 암호 입력

참고: https://jung-family-nas.uk:9001/remote.php/dav/files/USERNAME에서 USERNAME은 로그인 시 사용되는 영문 이름을 입력해주세요.
